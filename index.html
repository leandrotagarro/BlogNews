Ops! Falha faz Microsoft recomendar Windows 11 para computadores incompatíveis

Um erro da Microsoft fez com que usuários de computadores incompatíveis com o Windows 11 recebessem alertas para fazer a atualização do Windows 10 para a versão mais recente do sistema operacional da empresa. O caso foi notado pelo usuário PhantonOcean2 no Twitter, que recebeu uma mensagem em tela cheia sugerindo que ele instalasse o Windows 11 em seu computador com 2 GB de RAM e sem compatibilidade com o chip Trust Plataform Module (TPM).
